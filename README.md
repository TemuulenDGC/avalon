## Getting Started Guide
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/TemuulenDGC/avalon.git
   cd avalon
   ```
2. **Install Dependencies**:
   Follow the instructions provided for your environment.
   ```bash
   npm install
   ```
3. **Run the Application**:
   ```bash
   npm start
   ```
